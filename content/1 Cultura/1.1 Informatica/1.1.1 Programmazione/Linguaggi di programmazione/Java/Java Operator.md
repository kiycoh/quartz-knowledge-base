# Operatori Java
È possibile classificazione gli operatori Java in base al tipo di operazione che eseguono:
- [[Operatori aritmetici Java]]: eseguono calcoli matematici basilari;
- [[Operatori di assegnazione Java]]: assegnano un valore ad una [[variabile Java]];
- [[Operatori di comparazione Java]]: confrontano due valori e restituiscono un valore booleano (`true`o `false`);
- [[Operatori logici]]: combinano due o più espressioni booleane;
- [[Operatori bitwise Java]]: eseguono operazioni manipolando singoli bit (0 e 1).

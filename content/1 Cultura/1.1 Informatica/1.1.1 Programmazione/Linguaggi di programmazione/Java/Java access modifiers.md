# Java access modifiers
Un **modificatore di accesso Java** è una [[Java keyword|keyword]]. Possiamo riconoscere:
- [[public (Modificatore Java)]]: il membro è accessibile da qualsiasi altra classe, indipendentemente dal package.
- [[private (Modificatore Java)]]: il membro è accessibile all'interno della classe dichiarata.
- [[protected (Modificatore Java)]]: il membro è accessibile dallo stesso pacchetto o sottoclasse.
- [[default (Modificatore Java)]]: il membro è accessibile solo dalla propria classe.

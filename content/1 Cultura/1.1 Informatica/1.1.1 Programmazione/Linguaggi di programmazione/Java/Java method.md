# Java method
Un **metodo Java** è un blocco di codice che viene eseguita ogni volta che viene richiamato, puoi passare parametri. È utile per riusare il codice, una volta definito può essere utilizzato più volte.
- Java fornisce dei [[Metodi predefiniti Java|metodi predefiniti]] come 

Un metodo può essere dichiarato in una classe con il nome del metodo e delle parentesi `System.out.println()` ma possono essere creati i propri metodi.

> 
> ```java
> public class Main {
>    static void Main(String[] args)
> ```

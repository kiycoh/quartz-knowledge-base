| Operator | Name           | Description                      | Example |
| -------- | -------------- | -------------------------------- | ------- |
| +        | Addition       | Somma due valori numerici        | x + y   |
| -        | Subtraction    | Sottrae un valore da un altro    | x - y   |
| *        | Multiplication | Moltiplica due valori            | x * y   |
| /        | Division       | Divide un valore da un altro     | x / y   |
| %        | Modulus        | Ritorna il resto della divisione | x % y   |
| ++       | Increment      | Incrementa il valore di 1        | ++x     |
| --       | Decrement      | Decrementa il valore di 1        | --x     |

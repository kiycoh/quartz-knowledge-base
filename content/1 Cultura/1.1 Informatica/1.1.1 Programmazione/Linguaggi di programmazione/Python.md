# Python
Python, creato nel 1991 da [Guido van Rossum](https://it.wikipedia.org/wiki/Guido_van_Rossum), ==è un [[Linguaggio di programmazione]] compilato ad alto livello, *[[Programmazione orientata agli oggetti (OOP)|orientato agli oggetti]]*==.
- [[Tipi di dati (Python)]]:`Numeric`, `Sequence`, `Mapping`, `Set`, `Boolean`.
- [[Funzione (Python)]]: le variabili stanno ad un valore come le funzioni stanno ad un blocco di codice.
- [[Loop (Python)]]: i cicli permettono di eseguire ripetutamente dei blocchi di codice senza che questi vengano riscritti.
- [[Eccezioni in Python]]: è possibile gestire gli errori noti in modo da non interrompere l'esecuzione del codice.
- [[Classe Python]]: una classe è un oggetto (simile ad un contenitore) che può contenere vari elementi tra cui metodi e attributi.
- [[Moduli Python]]: file contenente definizioni e istruzioni in codice.
- [[Librerie Python]]: una libreria è un insieme di moduli correlati.
- [[Lettura e scrittura su file in Python]], è possibile modificare ed utilizzare file come supporto alla programmazione.

> [!QUESTION] Da dove viene il nome *Python*?
> Python deve il proprio nome al gruppo comico [Monty Python](https://it.wikipedia.org/wiki/Monty_Python), di cui lo sviluppatore era un grande fan.

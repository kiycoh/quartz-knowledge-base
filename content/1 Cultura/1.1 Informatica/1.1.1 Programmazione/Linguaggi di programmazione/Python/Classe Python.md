# Classe Python 
Python supporta una [[Programmazione orientata agli oggetti (OOP)]]. ==Una classe consente di creare oggetti== al cui interno le variabili (di istanza) sono chiamati **attributi** e le funzioni come **metodi**.

Una classe è un "Abstract Data Type" e corrisponde ad un set di oggetti e operazioni sugli stessi. Un'oggetto è un'istanza della classe. Segue le [[Proprietà della programmazione OOP]]

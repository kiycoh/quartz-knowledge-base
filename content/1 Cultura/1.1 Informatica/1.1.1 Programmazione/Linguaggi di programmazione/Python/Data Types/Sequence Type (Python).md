# Python Sequence Type
In Queste classi sono ***collezioni ordinate di dati***, simili nella struttura generale ma diverse nello specifico ai ==*set* e *dictionaries*== che sono ***collezioni disordinate.***
- **[[List (Python)]]**: sequenza indicizzata di elementi (es. `.append`, `.index`, `.sort`);
- **[[Tuple]]**: sequenza indicizzata e immutabile di elementi (es. `.count`, `.index`);
- **[[String]]**: sequenza immutabile di caratteri (Extended ASCII) (es. `.count`, `.index`, `.join`, `.replace`, `.split`, `.strip`)

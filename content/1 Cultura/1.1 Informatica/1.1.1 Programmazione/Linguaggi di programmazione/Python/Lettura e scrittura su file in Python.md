# File di Testo:
   - **Apertura e Chiusura:**
     - L'apertura di un file è essenziale prima di eseguire operazioni di lettura o scrittura.
     - La chiusura è altrettanto importante per liberare le risorse e garantire la corretta elaborazione dei dati.

   - **Lettura di un File Riga per Riga:**
     - Il metodo `readline()` legge il file riga per riga, partendo dalla posizione corrente.
     - Fornisce una stringa fino al carattere di fine riga (`\n`).

   - **Scrittura su un File:**
     - La scrittura su un file avviene tramite metodi come `write` o `print` con l'argomento `file`.
     - È possibile controllare l'allineamento e la formattazione dei dati scritti.

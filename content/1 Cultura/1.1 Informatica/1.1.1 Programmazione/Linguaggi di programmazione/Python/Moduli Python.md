# Cos'è un modulo?
==Un modulo è un file contenente definizioni e istruzioni Python==. Possono contenere funzioni, classi, variabili. Quando si vuole organizzare un codice in unità logiche e riutilizzabili si creano moduli separati. `modulo.py`

# Cosa si intende per modularità?
La modularità si riferisce alla capacità di dividere il codice in Moduli o componenti separati. In Python, i moduli sono file contenenti definizioni e istruzioni Python. Un modulo può essere importato in altri file per riutilizzare il codice. Questo facilita la gestione e l'organizzazione del codice, migliorando la sua leggibilità e manutenibilità.

# Modificatori (SQL)

| **Attributo**                 | **Descrizione**                                                |
| ----------------------------- | -------------------------------------------------------------- |
| `AUTO_INCREMENT`              | Incrementa automaticamente il valore di una colonna (MySQL).   |
| `SERIAL`                      | Tipo speciale per colonne autoincrementate (PostgreSQL).       |
| `ON UPDATE CURRENT_TIMESTAMP` | Aggiorna automaticamente la colonna con il timestamp corrente. |

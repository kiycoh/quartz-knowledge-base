| **Attributo**                     | **Descrizione**                                                 |
| --------------------------------- | --------------------------------------------------------------- |
| `INT` / `INTEGER`                 | Numero intero di dimensione variabile.                          |
| `BIGINT`                          | Numero intero di dimensione maggiore.                           |
| `SMALLINT`                        | Numero intero di dimensione ridotta.                            |
| `TINYINT`                         | Numero intero di piccola dimensione (MySQL).                    |
| `DECIMAL(p, s)` / `NUMERIC(p, s)` | Numero con precisione e scala specificate.                      |
| `FLOAT`                           | Numero con virgola mobile.                                      |
| `REAL`                            | Tipo di numero a virgola mobile con precisione ridotta.         |
| `CHAR(n)`                         | Stringa a lunghezza fissa.                                      |
| `VARCHAR(n)`                      | Stringa a lunghezza variabile.                                  |
| `TEXT`                            | Testo di lunghezza illimitata.                                  |
| `DATE`                            | Data nel formato `YYYY-MM-DD`.                                  |
| `TIME`                            | Ora nel formato `HH:MM:SS`.                                     |
| `DATETIME` / `TIMESTAMP`          | Data e ora combinate.                                           |
| `BOOLEAN`                         | Tipo booleano (`TRUE` o `FALSE`).                               |
| `BLOB`                            | Dati binari di dimensione variabile (per immagini, file, ecc.). |
| `JSON`                            | Dati in formato JSON (supportato in MySQL, PostgreSQL, ecc.).   |
| `UUID`                            | Identificatore univoco universale (PostgreSQL, altri).          |

# Definizione di Machine Learning
L'**apprendimento automatico** (o in inglese **machine learning**) è un *approccio simbolico* all'[[Intelligenza artificiale]].
> "Il passato può influenzare il futuro"
 
> [!NOTE] Approccio *simbolico vs sub-simbolico*
> * Un *sistema simbolico* è una **rappresentazione esplicita del sapere** (simboli, regole, concetti, ..);
> * Un *sistema sub-simbolico* è una **rappresentazione distribuita del sapere**, l'informazione è rappresentata in modo 

## Tipi di apprendimento
* **[[Apprendimento supervisionato]]**: un algoritmo di ML che utilizza **dati etichettati** per predire una o più categorie rilevate dal dataset.
* **[[Apprendimento non supervisionato]]**: un algoritmo di ML che utilizza **dati non etichettati** per predire una o più categorie rilevate dal dataset.

	* [[Classificazione (Machine learning)]]

## Algoritmo del gradiente discendente
L'algoritmo (un modello dati determinati parametri) che permette di mediare la retta che divide le caratteristiche inserite nel piano.
* **Misura di performance**

$$
\sum_{i=x}^n | \text{segno} (x_iN-i)
$$

# Data Definition Language (SQL)

- Definisce lo schema della base di dati.
- Permette di creare, cancellare e modificare tabelle, viste, indici e altri oggetti del database.
- Esempi di comandi DDL: `CREATE TABLE`, `ALTER TABLE`, `DROP TABLE`, `CREATE VIEW`.

# Connettivi logici (Informatica Teorica)

| Simbolo | Nome                                        | Esempio                                                   |
| ------- | ------------------------------------------- | --------------------------------------------------------- |
| ∧       | AND logico (congiunzione)                   | A ∧ B = vero solo se A e B sono entrambi veri             |
| ∨       | OR logico (disgiunzione)                    | A ∨ B = vero se almeno uno tra A o B è vero               |
| ¬       | NOT logico (negazione)                      | ¬A = vero se A è falso                                    |
| →       | Implicazione logica                         | A → B = falso solo se A è vero e B è falso                |
| ↔       | Doppia implicazione logica (bicondizionale) | A ↔ B = vero se A e B sono entrambi veri o entrambi falsi |
| ⊕       | OR esclusivo (XOR)                          | A ⊕ B = vero se A e B sono diversi                        |
| ⊤       | Tautologia (vero)                           | Sempre vero                                               |
| ⊥       | Contraddizione (falso)                      | Sempre falso                                              |

| Protocollo                                   | Descrizione                                                                                                                                       |
| -------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| [[HyperText Transfer Protocol (HTTP)\|HTTP]] | Usato da web browsers e server per comunicare e scambiare pacchetti                                                                               |
| HTTPS                                        | Estensione di HTTP, più sicuro e garantisce comunicazione crittografata                                                                           |
| FTP                                          | Usato per trasferire files da client a server                                                                                                     |
| [[Transmission Control Protocol (TCP)\|TCP]] | Consegna un flusso di bytes ordinati da un computer ad un altro                                                                                   |
| UDP                                          |                                                                                                                                                   |
| [[Internet Protocol (IP)\|IP]]               |                                                                                                                                                   |
| SMTP                                         | Usato per trasmettere emails tra IP                                                                                                               |
| SSH                                          | Protocollo crittografico per trasmissione sicura di dati, login remoto ed esecuzione comandi da remoto tra due computer connessi alla stessa rete |
Esistono anche combinazioni tra protocolli:
 * TCP e IP costituiscono il protocollo **TCP/IP**, che è la base delle comunicazioni su [[Internet]].

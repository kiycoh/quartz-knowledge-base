# Array
Un'array è una delle [[Struttura dati (Informatica teorica)|strutture dati lineari]] più semplici, che memorizzano una sequenza contigua di byte in memoria. Gli elementi sono accessibili tramite un indirizzamento diretto basato sull'indice (molto efficiente). In breve, un array è una [[Struttura dati lineare]] per la sua disposizione logica e **statica** per la sua allocazione di memoria.
- Gli arrays permettono di processare grandi quantità di dati in un tempo relativamente breve.
	- [[Operazioni su array]]
- ==Gli elementi che compongono un'array sono indicizzati==, ciò semplifica l'identificazione di ciascun elemento.
- Un'array può implementare altre strutture dati come stacks, [[Coda (Struttura dati)|code]], heaps, [[Hash table|hash-tables]], ecc.. 

>![[Array (informatica)_image_1.png]]
> Ipotizzando un modello RAM classico, un array di dimensione `n` occupa uno spazio continuo in memoria. L'indirizzo di un elemento `i` è calcolabile direttamente.
## Utilizzo pratico degli arrays
Gli arrays possono essere utilizzati in varie applicazioni:
- Utili per risolvere problemi con [[Matrice (Struttura dati)|matrici]].
- Possono essere implementati dei [[Database|databases]].
- Utili per l'implementazione di [[Algoritmo di ordinamento]]
- Utili negli algoritmi di processamento del linguaggio dove ogni parola è un segnale nell'array.
- Lo schermo di un computer funziona tramite l'uso di un [[Array (NumPy)|array multidimensionale]].
- Utili in molteplici sistemi di gestione dati (librerie, università, parlamento, ecc..)
- Utili nell'immagazzinare informazioni per computazioni matematiche.
- Utili nel processare immagini.

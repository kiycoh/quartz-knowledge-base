- **Descrizione**: Estensione bidimensionale (o multidimensionale) degli array. Sono rappresentate usando array monodimensionali.
- **Rappresentazione**:
        - **Per righe**: Ogni riga è un array monodimensionale.
        - **Per colonne**: Ogni colonna è un array monodimensionale.
        - La posizione di un elemento `M[i][j]` è determinata da formule specifiche per riga o per colonna.
- **Flessibilità**: Poco flessibili se si hanno righe o colonne con numero di elementi differente; in questo caso si può usare una rappresentazione con un array di puntatori ad altri array.    - **Utilizzo Pratico**: Problemi di calcolo matriciale (es. prodotto a catena di matrici).

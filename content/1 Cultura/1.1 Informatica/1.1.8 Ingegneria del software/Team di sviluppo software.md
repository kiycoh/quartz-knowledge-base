# Cos'è un team di sviluppo software?



> [!QUESTION] Cosa fa un team di sviluppo?
> Un **team di sviluppo** risponde ai seguenti interrogativi:
>  * Come progettare il prodotto software? $\to$ Architetto software
>  * Come costruire il prodotto? $\to$ Programmatore
>  * A cosa serve il prodotto? $\to$ Esperti di dominio
>  * Come va costruita l'interfaccia? $\to$ UI designer
>  * Come controllare la qualità del prodotto? $\to$ Tester
>  * Come usare le risorse di progetto? $\to$ Project manager
>  * Come riusare il software esistente? $\to$ Gestori delle configurazioni
